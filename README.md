## Dataset
Dataset simulado con 30 registros de ventas.
Columnas: id, producto, cantidad, precio, fecha, monto_total, mes.
Almacenado en: datos/ventas.csv

## Resultados obtenidos
- Ventas totales: $22,635.00
- Producto mas vendido: Mouse
- Grafico de evolucion mensual guardado en: resultados/grafico_ventas.png

## Como ejecutar
1. Abrir Google Colab
2. Clonar este repositorio
3. Ejecutar scripts/analisis_ventas.py
4. Los resultados se guardan en /resultados

## Estructura del repositorio
- /datos - Dataset CSV de ventas
- /scripts - Script de analisis en Python
- /resultados - Grafico generado
